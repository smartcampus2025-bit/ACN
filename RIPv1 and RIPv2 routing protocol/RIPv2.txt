On Router R2, execute the debug IP rip command to check the RIPv2 configuration and you will see a log
R2(config)# do debug ip rip

Do after debug
R2(config)# do undebug ip rip

We can use the Wireshark program to verify RIPv2.Start the Wireshark program I in between Serial
communication line. We can view the RIP V2 (224.0.0.9) outpu
Step 1:-  right click on router 2
Step 2:-  Click on Start capture
In this screen we can see RIPv2 output on the destination 224.0.0.
